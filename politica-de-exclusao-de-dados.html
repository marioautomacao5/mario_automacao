Guia de Exclusão de Dados

Seu Negócio Digital

Última atualização:

1. Objetivo

Esta página serve para orientar clientes sobre como solicitar a exclusão de dados pessoais coletados por nossas soluções digitais, como landing pages, chatbots ou formulários de captura de leads, garantindo conformidade com a LGPD.

2. Solicitação de exclusão de dados

Os clientes podem solicitar a exclusão de dados de duas maneiras:

  • Enviando uma requisição formal ao nosso suporte via e-mail: suporte@seudominio.com
  • Utilizando o endpoint de exclusão disponibilizado em suas integrações (para desenvolvedores e sistemas automatizados).

3. Exemplo de endpoint para exclusão

Para clientes que usam chatbots ou APIs, disponibilizamos o seguinte endpoint:

DELETE /api/leads/{email}

Substitua {email} pelo e-mail do lead que deseja excluir. Esta chamada remove o registro permanentemente do banco de dados.

Exemplo de requisição via JavaScript (fetch):

fetch('https://seudominio.com/api/leads/cliente@exemplo.com', {
    method: 'DELETE',
    headers: { 'Content-Type': 'application/json' }
})
.then(response => {
    if(response.ok) {
        alert('Lead excluído com sucesso!');
    } else {
        alert('Falha ao excluir lead.');
    }
})
.catch(error => console.error(error));

4. Confirmação e logs

Após a exclusão, será gerado apenas um registro de ação (log), sem armazenar os dados pessoais do lead, garantindo rastreabilidade e conformidade legal.

5. Prazo de atendimento

Todas as solicitações de exclusão serão processadas dentro de 5 dias úteis após recebimento da requisição.
