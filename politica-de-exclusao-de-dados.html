Guia de Exclusão de Dados

Seu Negócio Digital

Última atualização:

1. Objetivo

Esta página serve para orientar clientes sobre como funciona a exclusão de dados pessoais coletados por nossas soluções digitais, como landing pages, chatbots ou formulários de captura de leads, garantindo transparência e conformidade com a LGPD.

2. Responsabilidade da exclusão

A exclusão de dados é de responsabilidade do administrador do sistema de cada projeto. Todo projeto que captura dados possui um painel administrativo (“Admin Panel”) que permite gerenciar e excluir leads diretamente.

Portanto, quaisquer solicitações de exclusão devem ser feitas diretamente ao usuário administrador do sistema associado ao projeto.

3. Como o administrador pode excluir dados

O painel administrativo permite ao usuário administrador:

  • Visualizar todos os leads capturados
  • Buscar leads por e-mail ou ID
  • Excluir leads individualmente ou em lote
  • Gerar logs da ação de exclusão para comprovação de conformidade

4. Endpoint de exclusão (para integração)

Para administradores que usam APIs ou chatbots integrados, existe um endpoint seguro disponível:

DELETE /api/leads/{email}

Substitua {email} pelo e-mail do lead que deseja excluir. Esta ação remove permanentemente o registro do banco de dados.

5. Prazo de atendimento

Após solicitação de exclusão ao administrador, os dados devem ser removidos imediatamente via painel ou conforme procedimento interno do projeto, garantindo eficiência e conformidade legal.

6. Contato

Se houver dúvidas sobre como solicitar a exclusão, entre em contato diretamente com o administrador do sistema do projeto ou com nosso suporte geral:

Email do suporte: suporte@seudominio.com
