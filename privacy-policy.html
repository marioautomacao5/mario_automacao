Política de Privacidade

Digital Cashier

Última atualização:

1. Introdução

O Digital Cashier valoriza a privacidade e a proteção de dados pessoais. Esta Política de Privacidade descreve como coletamos, utilizamos, armazenamos e protegemos as informações fornecidas pelos usuários que interagem com nossos chatbots através do WhatsApp, em conformidade com a Lei nº 13.709/2018 (Lei Geral de Proteção de Dados – LGPD).

2. Dados pessoais coletados

Durante a interação com o chatbot, poderão ser coletados os seguintes dados pessoais:

  • Nome do usuário
  • Número de telefone
  • Mensagens enviadas durante a conversa
  • Informações fornecidas voluntariamente (ex: interesse em produtos, orçamento, localização, e-mail, empresa, etc.)
  • Dados necessários para qualificação de leads comerciais

3. Finalidade do tratamento dos dados

Os dados pessoais coletados são utilizados para as seguintes finalidades:

  • Identificação de potenciais clientes (leads)
  • Gestão de funil de vendas para empresas que utilizam o Digital Cashier
  • Atendimento automatizado via WhatsApp
  • Melhoria da experiência do usuário
  • Análises estatísticas e melhoria contínua do serviço

4. Compartilhamento de dados

Os dados coletados poderão ser compartilhados exclusivamente com a empresa responsável pelo atendimento ao usuário, que utiliza o Digital Cashier como ferramenta de automação comercial. O Digital Cashier não comercializa dados pessoais com terceiros.

5. Base legal para tratamento de dados

O tratamento dos dados pessoais é realizado com base nas seguintes hipóteses legais previstas na LGPD:

  • Consentimento do titular dos dados
  • Execução de contrato ou procedimentos preliminares relacionados ao atendimento solicitado
  • Legítimo interesse para melhoria dos serviços e atendimento comercial

6. Armazenamento e segurança dos dados

Adotamos medidas técnicas e organizacionais adequadas para proteger os dados pessoais contra acessos não autorizados, perda, alteração ou divulgação indevida.

7. Direitos do titular dos dados

Nos termos da LGPD, o usuário poderá solicitar a qualquer momento:

  • Confirmação da existência de tratamento de dados
  • Acesso aos dados pessoais
  • Correção de dados incompletos ou incorretos
  • Exclusão de dados pessoais
  • Revogação do consentimento

8. Uso da API do WhatsApp

O Digital Cashier utiliza a API oficial do WhatsApp Business para processamento de mensagens. Ao interagir com o chatbot, o usuário também estará sujeito às políticas de privacidade da plataforma do WhatsApp.

9. Alterações nesta política

Esta Política de Privacidade poderá ser atualizada a qualquer momento para garantir conformidade legal ou melhorias no serviço.
